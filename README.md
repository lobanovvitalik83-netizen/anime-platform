# Stage 14 - Media Cards Unified UI

В архиве:
- весь stage 13
- новая вкладка "Медиа" вместо служебных разделов в навигации
- быстрый доступ к созданию карточки
- упрощённая форма карточки:
  - жанр (аниме / сериал / фильм)
  - название
  - сезон
  - серия
  - фото / постер / видео
  - загрузка файлом или внешняя ссылка
  - статус
  - автогенерация кода
- убраны из карточки:
  - оригинальное название
  - год
  - название сезона
  - название серии
  - описание карточки
  - описание серии
- список карточек "Медиа"
- массовый выбор карточек:
  - удалить выбранные
  - открыть выбранные в редактировании

Новые страницы:
- /admin/media
- /admin/media/{title_id}/edit
- /admin/card-builder
