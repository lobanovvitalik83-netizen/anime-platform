# Stage 20 - Analytics / Audit / Lookup Stats

В архиве:
- весь stage 19
- аналитика найденных и ненайденных кодов
- счётчик попыток по каждому коду
- время последнего обращения к коду
- список последних попыток поиска
- просмотр конкретного кода
- карточка, связанная с кодом, даже если код не был найден как активный
- аудит действий админов в одном разделе аналитики

Новые страницы:
- /admin/analytics
- /admin/analytics/codes/{code}

Что проверять:
- найденные коды
- ненайденные коды
- сколько раз был запрос по коду
- когда был последний запрос
- открытие детальной страницы кода
- показ карточки под кодом, если код существует, но не активен / не найден
