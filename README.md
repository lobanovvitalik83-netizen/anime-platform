# Stage 16 - Roles / Editors / Access

В архиве:
- весь stage 15
- роли: superadmin / admin / editor
- страница управления командой
- создание пользователя
- редактирование роли
- смена пароля
- активация / деактивация аккаунта
- ограничения по ролям в админке

Права:
- editor: медиа, создание карточек, редактирование карточек, lookup
- admin: всё editor + удаление карточек, коды, импорт/экспорт
- superadmin: всё admin + команда и роли

Новые страницы:
- /admin/team
- /admin/team/new
- /admin/team/{id}/edit
