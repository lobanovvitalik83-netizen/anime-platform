# Stage 26 - Unified admin operations

Что реально добавлено в этом архиве:
- фикс режима Telegram бота:
  - кнопки «Поиск по коду / Репорт / Помощь»
  - поиск по коду снова работает
  - репорты создаются только в режиме «Репорт»
- уведомления в админке:
  - новые репорты
  - новые сообщения в чатах
- отдельная страница уведомлений:
  - /admin/notifications
- улучшена вкладка Медиа:
  - карточки удобнее читать
  - превью внешних изображений
  - быстрые действия
- улучшена вкладка Коды:
  - понятные подписи
  - быстрые ссылки на топ найденных / ненайденных
  - summary по статусам
- приватные API docs только для superadmin:
  - /admin/api-docs
- assistant role и человеческие названия разрешений уже встроены и сохранены
- расширенная аналитика, admin actions и advanced import/export остаются доступны из предыдущего этапа

Что НЕ выдумано:
- архив не обещает websocket-реалтайм и не меняет storage-архитектуру
- это этап стабилизации и удобства поверх существующего stage 25
