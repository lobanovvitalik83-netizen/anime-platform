Основные данные

Медиа

1. Загрузить файл прямо в Яндекс Диск

Файл сразу уходит во внешний storage. В BotHost остаются только path, URL и metadata.

2. Импортировать по прямому URL с копированием в storage

Система скачает файл по ссылке и перельёт его в твой внешний storage. Для share-страниц без прямой отдачи файла может потребоваться прямая download-ссылка.

3. Просто использовать внешнюю ссылку без копирования

В этом режиме файл не копируется в твой storage. При удалении карточки удаляется только связь, а не исходный файл.
