- в подписи только:
  - название
  - жанр
  - сезон
  - серия
- служебные поля убраны

2. VK callback:
- /api/vk/callback
- confirmation token
- message_new
- anti-spam по event_id
- plain-text ответы
- репорты
- поиск по коду

3. Reset password by link:
- POST /admin/team/{id}/create-reset-link
- GET/POST /admin/reset-password/{token}
- ссылка одноразовая и имеет TTL 24 часа

4. Achievements:
- /admin/achievements
- /admin/achievements/new
- /admin/achievements/{id}/edit
- /admin/people/{id}/achievements/grant
- отзыв ачивки
- бейджи в профиле и в карточке сотрудника
